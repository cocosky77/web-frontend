Web Top 이란?

웹탑은 컴퓨터 운영체제와 인터넷을 통합하는 형태로, 파일관리, 정보관리 등에 있어 둘을 유기적으로 연결시킨 것이다. 웹탑 환경 아래서 사용자가 컴퓨터를 조작하지 않고도 사전에 지정해 놓은 채널에서 제공하는 정보를 검색할 수 있도록 하는 서비스를 웹탑서비스라 한다.

웹 데스크톱(web desktop) 또는 웹톱(webtop)은 웹 브라우저나 유사 클라이언트 애플리케이션에 임베디드된 데스크톱 환경이다. 웹톱은 웹 애플리케이션, 웹 서비스, 클라이언트-서버 애플리케이션, 웹 애플리케이션 서버, 그리고 로컬 클라이언트의 애플리케이션들을 데스크톱 메타포를 사용하여 데스크톱 환경으로 통합한다.

웹 데스크톱은 윈도우, 맥, 그리고 유닉스 및 리눅스 시스템의 그래픽 사용자 인터페이스와 유사한 환경을 제공한다. 웹톱에서 애플리케이션, 데이터, 파일, 구성, 설정, 접근 권한은 네트워크상에 원격으로 보관한다. 컴퓨팅의 많은 부분이 원격으로 처리된다. 브라우저가 주로 화면 출력과 입력 목적으로 사용된다.

"웹 데스크톱"과 "웹톱"이라는 용어는 TinyOS 등의 네트워크 운영 체제, 그리고 인페르노 등 분산 운영 체제와는 구별된다. 웹 운영 체제와 구별된다. 웹 데스크톱은 웹 운영 체제, WebOS를 의미하는 경우로 잘못 사용되기도 한다.

웹톱은 사용자가 직접 컴퓨터를 조작하지 않고서도 미리 지정해 놓은 원격 인터넷 사이트에서 밀어내는 여러 정보를 검색할 수 있도록 해주는 서비스이다. 차세대 인터넷 핵심기술 중 가장 중요한 것이 푸시(Push) 기술인데 인터넷이 정보를 밀어서 이용자에게 제공하는 기술이다.

인터넷 익스플로러 4.0이 윈도우 95와 연결되면서 등장한 개념으로 액티브 데스크 탑이라는 이름으로 마이크로소프트사가 먼저 선보였지만 넷스케이프도 최근 버전에서 '오로라'라는 웹탑을 적용하고 있다. 웹탑은 사용자의 데스크탑 월 페이퍼를 인터랙티브 채널로 바꾸어 주는 환경으로 인터넷과 운영체제가 유기적으로 연결된 형태만을 의미하기 때문에 그것을 위한 기술은 서로 다를 수 있다. 따라서 윈도우뿐 아니라 매킨토시, 유닉스 환경까지도 적용되며 계속 발전해 갈 것으로 보인다.

JAVA 애플리케이션의 설치가 간단해지면서 웹탑의 개념이 확대되었는데, 웹탑에서는 네트워크 컴퓨팅 환경 전체를 직원들이 필요로 할 때마다 JAVA 클라이언트로 다운로드할 수 있다. 웹탑이 조직에 미치는 영향은 지대하여, 데스크탑 지원비용은 크게 줄어들고 데이터는 중앙 서버에 안전하게 보존되는 한편, 한번 다운로드된 JAVA 애플리케이션은 데스크탑상에서 국지적으로 실행되므로 네트워크와 서버 이용률 증가를 막을 수 있다.

또 자체 홈페이지를 위한 URL만 이동하면 되므로 떨어져 있는 사무실간에 직원의 컴퓨터나 데이터를 이동하는 데 따른 비용이 전혀 필요 없다. 웹탑은 또 사내공고, 메시지, 교육 등이 모두 직원 홈페이지의 고정된 위치에 제공되므로 직원들간의 통신의 장으로 이용될 수 있으며, 직원들이 표준 브라우저가 설치되어 있는 가정용 PC나 기업 웹탑에 안전하게 억세스할 수 있기 때문에 원격 통신도 단순해진다. 그러나 통합기능으로 인한 컴퓨터 속도의 저하와 인터넷에 연결하기가 쉽지 않게 될 모뎀 사용자들의 불편을 어떻게 해결할 것인가는 풀어야 할 과제이다.
